Summary

A small inline label for status, categorization, or counts. Five semantic variants via attribute.


            <r-badge primary>New</r-badge>
            <r-badge success>Active</r-badge>
Variants
Default Primary Success Warning Danger Neutral
Attributes
  • primary — blue filled pill
  • success — green filled pill
  • warning — yellow filled pill (dark text for contrast)
  • danger — red filled pill
  • neutral — gray filled pill
 CSS Variables
--badge-font-size: var(--text-xs)
--badge-font-weight: var(--font-weight-medium)
--badge-padding: 0 var(--space-2)
--badge-radius: 9999px — pill shape by default
--badge-height: 20px
--badge-color: text color (defaults to bg color for contrast)
--badge-bg: background fill color
