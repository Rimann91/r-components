Summary

Wraps any element and shows a tooltip on hover or focus. Pure CSS — no JS measurement. Position defaults to top.


            <r-tooltip content="Helpful text">
              <r-button primary>Hover me</r-button>
            </r-tooltip>
Positions
Top Bottom Left Right
With other components
Download Beta
Attributes
  • content — tooltip text
  • position — top (default) | bottom | left | right
 CSS Variables
--tooltip-font-size: var(--text-xs)
--tooltip-padding: var(--space-1) var(--space-2)
--tooltip-radius: var(--radius)
--tooltip-bg: var(--color-fg-dark) — dark surface
--tooltip-color: var(--color-bg-primary) — light text
--tooltip-max-width: 240px
--tooltip-transition: var(--transition-fast)
--tooltip-gap: var(--space-1) — distance from trigger
