A zero-dependency, framework-free web component primitive library built with the Gruvbox colour theme and Fira Code monospace font.

Select a component from the sidebar to view its documentation and live demos.

 Primitives

Single-responsibility components: buttons, inputs, feedback, overlays.

 Layout

Structural wrappers: stack, cluster, grid, container.

 Design Tokens

All components are built on a shared token foundation in styles/gruvbox_dark_colors.css:

  • Colour palette (Gruvbox dark/light)
  • Spacing scale — --space-1 through --space-12
  • Typography scale — --text-xs through --text-4xl
  • Border radius — --radius
  • Shadows — --shadow-sm / --shadow-md / --shadow-lg
  • Transitions — --transition-fast / --transition-base / --transition-slow

Every component exposes --component-* CSS custom properties on :host for per-instance overrides without piercing the shadow DOM.
