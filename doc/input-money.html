Summary

A monetary input with a currency symbol prefix. Formats to two decimal places on blur. Accepts only numeric input. Symbol defaults to $ and can be changed via the symbol attribute.


            <r-input-money label="Price" value="9.99"></r-input-money>
            <r-input-money label="Amount" symbol="€"></r-input-money>
Demo
Attributes
  • label — text above the input
  • value — initial value (formatted to 2dp on load)
  • symbol — currency symbol shown as prefix (default: $)
  • disabled — disables the input
  • error — activates error styles
  • error-message — error text shown below when error is set
 Events
  • change — fires on input with detail: \{ value: number \}
