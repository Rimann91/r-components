Summary

A simple visual separator. Renders horizontally by default; add the vertical attribute for a vertical line.


            <r-divider></r-divider>
            <r-divider vertical></r-divider>
Horizontal (default) Above the divider Below the divider Vertical
Left Right
Attributes
  • vertical — renders as a vertical line (height: 100% of parent)
 CSS Variables
--divider-color: var(--color-divider)
--divider-thickness: 1px
--divider-margin: var(--space-2) 0 — applies to horizontal only
