Summary

A styled text input with a label, placeholder, error state, and disabled state.


            <r-input-text label="Username" placeholder="Enter username"></r-input-text>
            <r-input-text label="Email" error error-message="Invalid email"></r-input-text>
            <r-input-text label="Disabled" disabled></r-input-text>
Demo
Attributes
  • label — text shown above the input
  • placeholder — placeholder text inside the input
  • value — sets the input value
  • disabled — disables the input (reduced opacity, not-allowed cursor)
  • error — activates error styles (red border and label)
  • error-message — error text shown below the input when error is set
