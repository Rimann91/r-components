Summary

A date input with a calendar popup. Click the 📅 button or the field to open. Navigate months with ‹ › arrows. Fires a change event with an ISO date string (YYYY-MM-DD).


            <r-input-date label="Start date"></r-input-date>
            <r-input-date label="Prefilled" value="2026-04-10"></r-input-date>
Demo
Attributes
  • label — text above the input
  • value — initial date (YYYY-MM-DD format)
  • disabled — disables the input
  • error — activates error styles
  • error-message — error text shown below when error is set
 Events
  • change — fires on day selection with detail: \{ value: "YYYY-MM-DD" \}
 CSS Variables
--date-radius: var(--radius)
--date-bg: var(--color-bg-primary)
--date-color: var(--color-fg-primary)
--date-border-color: var(--color-divider)

--date-today-color: var(--color-primary-blue) — today highlight

--date-popup-shadow: var(--shadow-base)
