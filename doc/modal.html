Summary

A dialog overlay built on the native <dialog> element. Closes on ✕ button, backdrop click, or Escape key. Supports a footer slot for action buttons.


            <r-modal id="my-modal" title="Confirm">
              Are you sure?
              <div slot="footer">
                <r-button danger>Delete</r-button>
              </div>
            </r-modal>
            <r-button primary onclick="document.getElementById('my-modal').open()">Open</r-button>

Attributes
  • open — presence shows the modal
  • title — text in the modal header
 Slots
  • default — modal body content
  • footer — action buttons shown below a divider
 Methods
  • open() — opens the modal, fires open event
  • close() — closes the modal, fires close event
 Events
  • open — fires when modal opens
  • close — fires when modal closes (any method)
