Summary

A styled radio button with label. Group radios using a shared name attribute. Fires a change event on selection.


            <r-radio name="fruit" value="apple">Apple</r-radio>
            <r-radio name="fruit" value="pear">Pear</r-radio>

Attributes
  • checked — sets initial checked state
  • disabled — disables interaction
  • name — groups radio buttons together
  • value — value passed in the change event
 Events
  • change — fires on selection with detail: \{ checked: boolean, value: string \}
