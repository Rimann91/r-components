Summary

An animated show/hide container driven by the open attribute. Uses a CSS grid trick for smooth height transitions with no JS measurement.


            <r-collapse open> Content </r-collapse>
Demo
Open Close Toggle
This content is inside the collapse. It animates smoothly in and out.
Attributes
  • open — presence of this attribute shows the content
 Methods
  • open() — sets the open attribute
  • close() — removes the open attribute
  • toggle() — toggles the open attribute
 CSS Variables
--collapse-transition: var(--transition-base) — animation speed and easing
