Summary

A numeric input with increment/decrement buttons. Browser spinners are hidden in favour of the custom ± controls. Buttons disable automatically at min and max boundaries.


            <r-input-number label="Qty" value="1" min="0" max="10"></r-input-number>
Demo
Attributes
  • label — text above the input
  • value — initial value
  • min — minimum allowed value (disables − at boundary)
  • max — maximum allowed value (disables + at boundary)
  • step — increment amount (default: 1)
  • disabled — disables the entire control
  • error — activates error styles
  • error-message — error text shown below when error is set
 Events
  • change — fires on value change with detail: \{ value: number \}
