Summary

A fixed-position toast notification that slides in from the bottom-right. Auto-dismisses after a configurable duration. Multiple toasts stack automatically.

Use the static RToast.show() API — no need to place elements in your HTML.


            RToast.show({ message: 'Saved!', variant: 'success' });
            RToast.show({ message: 'Uh oh!', variant: 'danger', duration: 0 }); // no auto-dismiss

Static API

Import toast.js and call:


            RToast.show({
              message:  'Your message here',
              variant:  'success',  // 'info' | 'success' | 'warning' | 'danger' | ''
              duration: 4000,       // ms until auto-dismiss. 0 = never
            });

Returns the r-toast element so you can dismiss it programmatically:


            const t = RToast.show({ message: 'Loading…', duration: 0 });
            // later...
            t._dismiss();
Variants
  • info — blue border/text
  • success — green border/text
  • warning — yellow border/text
  • danger — red border/text
  • (none) — default border/text
