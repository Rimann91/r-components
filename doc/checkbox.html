Summary

A styled checkbox with label. Fires a change event with detail.checked on toggle.


            <r-checkbox>Accept terms</r-checkbox>
            <r-checkbox checked>Pre-checked</r-checkbox>
            <r-checkbox disabled>Disabled</r-checkbox>
Demo
Unchecked Pre-checked Disabled Checked + disabled
Attributes
  • checked — sets initial checked state
  • disabled — disables interaction
  • name — passed to the underlying input
  • value — passed to the underlying input
 Events
  • change — fires on toggle with detail: \{ checked: boolean \}
 CSS Variables
--checkbox-size: 16px
--checkbox-radius: calc(var(--radius) / 2)
--checkbox-color: var(--color-primary-blue) — checked fill
--checkbox-bg: var(--color-bg-primary)
--checkbox-border-color: var(--color-divider)
--checkbox-focus-ring: var(--focus-ring)
--checkbox-gap: var(--space-2) — gap between box and label
