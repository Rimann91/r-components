Summary

A date + time input with a calendar popup. Pick a date from the calendar, enter the time in the HH:MM fields, then click Confirm. Fires a change event with a full ISO timestamp.


            <r-input-datetime label="Scheduled at"></r-input-datetime>
Demo
Attributes
  • label — text above the input
  • value — initial datetime (ISO 8601 string)
  • disabled — disables the input
  • error — activates error styles
  • error-message — error text shown below when error is set
 Events
  • change — fires on confirm with detail: \{ value: ISO string \}
 CSS Variables
--datetime-radius: var(--radius)
--datetime-bg: var(--color-bg-primary)
--datetime-color: var(--color-fg-primary)
--datetime-border-color: var(--color-divider)
--datetime-accent: var(--color-primary-blue) — selected day and confirm button
--datetime-today-color: var(--color-primary-blue)
