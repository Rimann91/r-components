Summary

A contextual feedback message. Four semantic variants. Optionally dismissible — clicking ✕ fires a dismiss event and removes the element.


            <r-alert info>Something to note.</r-alert>
            <r-alert danger dismissible>Destructive action taken.</r-alert>
Variants
Default — no variant Info — informational message Success — action completed Warning — proceed with caution Danger — something went wrong
Dismissible
This alert can be dismissed. Click ✕ to remove it. Danger! This is also dismissible.
Attributes
  • info — blue border/text
  • success — green border/text
  • warning — yellow border/text
  • danger — red border/text
  • dismissible — shows ✕ button; fires dismiss event on click and removes element
 CSS Variables
--alert-font-size: var(--text-sm)
--alert-padding: var(--space-3) var(--space-4)
--alert-radius: var(--radius)
--alert-color: text color
--alert-bg: background color
--alert-border-color: border color
--alert-border-width: 1px
