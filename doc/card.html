Summary

Card is the most basic content container. It provides a background, bordered surface with an optional titled header area separated by a divider.

Examples Basic card with title Default slot content goes here. 
<r-card title="Card Title">
  Content goes here.
</r-card>
Card with action in header (header-slot-right)
Edit
Use slot="header-slot-right" to place content on the right side of the header. 
<r-card title="With Header Action">
  <r-button slot="header-slot-right" tertiary>Edit</r-button>
  Content goes here.
</r-card>
Card with no title (header hidden via slot="header")
Omitting the title attribute leaves the header bar visible but empty. Supply an empty slot="header" to hide it entirely. 
<r-card>
  <div slot="header"></div>
  Content with no header.
</r-card>
Attributes
  • title: Sets the card header title text.
 Slots
  • default: Main content area of the card.
  • header-slot-right: Content placed on the right side of the header (e.g. a button or badge).
  • slot-left: Replaces the title area in the header left side.
  • header: Replaces the entire header row. Use an empty element to hide the header completely.
 CSS Variables
--card-bg: var(--color-bg-primary) — card surface background color.
--card-border-color: var(--color-divider) — border and header divider color.
--card-radius: var(--radius) — corner radius.
--card-padding: var(--space-2) — padding for header and content areas.
--card-font-size: var(--text-base) — body font size.
--card-title-size: var(--text-xl) — header title font size.
--card-title-weight: var(--font-weight-bold) — header title font weight.
