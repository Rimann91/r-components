            <r-button primary>Text</r-button>
r-button provides five semantic styles accesible by attribute Attributes
  • primary: default to this for most button
  • secondary: usually for when you need a second button next to primary
  • tertiary: blends in with primary background
  • success: green color for affirmative actions or aknowledgment
  • danger: red color for destructive actions
